The permission-gating slice of an [agent mode](./Agent%20mode.md) — which [tool calls](./Tool%20call.md) trigger a [permission request](./Permission%20request.md) and which run automatically. The original purpose of mode systems before [harnesses](./Harness.md) started bundling behavioral instructions on top.

**中文：** [智能体模式](./Agent%20mode.md)的权限门控部分——哪些[工具调用](./Tool%20call.md)会触发[权限请求](./Permission%20request.md)，哪些自动运行。在[harnesses](./Harness.md)开始在之上捆绑行为指令之前，模式系统的原始目的。

*Usage:*

"It paused on every grep — totally killed the [AFK](./AFK.md) run."

**中文：** "它在每次 grep 时都暂停——完全毁掉了[AFK](./AFK.md)（离开键盘）运行。"

"Loosen the permission mode for read-only [tools](./Tool.md), keep prompting on writes and shell. Most permission requests on a research [session](./Session.md) are noise."

**中文：** "对只读[工具](./Tool.md)放宽权限模式，对写入和 shell 操作保持提示。研究[会话](./Session.md)中的大多数权限请求都是噪音。"
